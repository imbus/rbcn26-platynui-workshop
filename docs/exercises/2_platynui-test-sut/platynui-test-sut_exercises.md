# Test Cases for PlatynUI Test SUT

## Controls Page

Exercise 1 - Buttons: Click "Regular Button" and ensure that it is clicked.
Exercise 2 - CheckBox/ComboBox/Switch: Activate "Flight Mode" Switch. Ensure that your test only activate "Flight Mode" if it is off.
Exercise 3 - RadioButtons: Select "Large" Radio Button.
Exercise 4 - Date/Time: Select today's date, 16:30 and verify that it is displayed in the StatusBar (Footer).
